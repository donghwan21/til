1. 계산 복잡도 빅오 표기법

   - O(n):필요한 계산횟수가 입력크기 n과 비례할때
   - O(1):필요한 계산횟수가 입력크기 n과 무관할때

   - 계산 복잡도는 시간 복잡도와 공간 복잡도로 나눌 수 있다
     사칙연산= 시간복잡도

   ```python
           def sqare_root(n):
               s = 0
               for i in range(1, n+1):
                   s = i**2+s
               return s

           print(sqare_root(10))

           #시간 복잡도:O(n)
   ```
